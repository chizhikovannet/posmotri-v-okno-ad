# Проект «Посмотри в окно»

Интерактивный сервис для просмотра видео из разных городов России в разное время суток.

## ✨ Возможности

- **Просмотр видео** — основной блок с видео, которое можно запустить
- **Поиск по городу** — возможность найти видео из конкретного города
- **Фильтр по времени суток** — выбор видео для утра, дня или ночи
- **Список карточек** — удобный скроллируемый список доступных видео
- **Пагинация** — кнопка «Показать ещё» для загрузки дополнительных видео
- **Адаптивные состояния** — все интерактивные элементы имеют стили для :hover, :active и :focus-visible

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, кастомные свойства)
- JavaScript (взаимодействие с API, динамическая отрисовка)
- Работа с API (получение данных о видео)

## 🎯 Функциональность

### Основные элементы
- **Блок с видео** — крупный плеер для просмотра выбранного ролика
- **Форма поиска** — поля для ввода города и выбора времени суток
- **Список карточек** — превью всех доступных видео с описанием

### Интерактивность
- При загрузке страницы подгружаются первые 5 видео
- Клик по карточке меняет видео в основном плеере
- Поиск фильтрует видео по заданным параметрам
- При ошибках показывается соответствующий блок с сообщением
- Во время загрузки данных отображается прелоадер

## 🎨 Особенности вёрстки

- **Pixel Perfect** — точное соответствие макету
- **Кастомный скроллбар** — стилизован в соответствии с дизайном
- **Кастомные чекбоксы** — реализованы через псевдоэлементы
- **Обрезка текста** — длинные заголовки обрезаются с многоточием
- **Все состояния элементов** — :hover, :active, :focus-visible
- **Доступность** — навигация с клавиатуры, визуальные подсказки
